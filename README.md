# Image_Saver_BE
Backend web framework script built on Typescript and ExpressJS

### Application built on:-
- ExpressJS
- Typescript

### Description:-
It is to save the Images on cloud on MongoDB Atlas with user authentication.

A Complete Typescript file based application with production ready build.
- npm run dev -> will initiate development server.
- npm run build -> will initiate the production ready code by compiling the entire project with typescript compilier, that it will be stored on dist folder for production.
- npm start -> will initiate the server of production ready code from 'dist' folder.

### FlowChart of Application
1) Login/signup for the Application to upload the Images.
2) Upload the Images in MongoDB cloud Atlas.
3) Get all the Images based on the User token.
4) Delete all the Images based on the token and ID generated for the image at the time of uploaded.
